Hi-Lo Card Counting

Understanding the most popular counting system

What is Card Counting?

Card counting is a strategy used in blackjack to determine whether the next hand is likely to give an advantage to the player or to the dealer. It is a mathematical technique — not memorization of every card — that tracks the ratio of high to low cards remaining in the shoe.

When many high cards (10s, face cards, Aces) remain in the deck, the player has an advantage because blackjacks become more likely and the dealer is more likely to bust on stiff hands. When many low cards remain, the dealer has the advantage.

Hi-Lo Card Values

The Hi-Lo system assigns a point value to every card dealt:

2, 3, 4, 5, 6 +1 Low cards — their removal is good for the player
7, 8, 9 0 Neutral cards — minimal impact on advantage
10, J, Q, K, A -1 High cards — their removal is bad for the player

Running Count

The Running Count (RC) is the cumulative total of all card values seen so far. You start at zero when a new shoe begins and add or subtract as each card is revealed.

Example: Cards dealt so far: 5 (+1), K (-1), 3 (+1), 7 (0), A (-1), 4 (+1)
Running Count = +1 - 1 + 1 + 0 - 1 + 1 = +1

True Count

The True Count (TC) adjusts the running count for the number of decks remaining in the shoe. This gives a more accurate picture of the player's advantage.

Formula: True Count = Running Count / Decks Remaining

Example: Running Count = +6, Decks Remaining = 3
True Count = +6 / 3 = +2

The true count is what matters for betting and playing decisions. A true count of +2 with 4 decks left is more meaningful than a running count of +2 with all 6 decks remaining.

What "Advantage" Means

In standard blackjack, the house has a small edge (roughly 0.5% with basic strategy). Card counting can shift this edge in the player's favor when the true count is sufficiently positive.

  • TC <= 0: The house has the edge. Bet the minimum
  • TC +1: Roughly even. A small increase in bet size
  • TC +2 or higher: Player advantage. Increase bet size proportionally

This version uses a True Count Bet Ramp to size bets automatically:

TC <= 0 → 1 unit  ·  TC 1 → 2 units  ·  TC 2 → 4 units  ·  TC 3+ → 6 units

Important Considerations

  • Variance is real. Even with a mathematical advantage, short-term results can swing wildly. A positive expected value does not guarantee winning in any given session
  • Bankroll management is essential. You need a sufficient bankroll to weather losing streaks. The general guideline is at least 100–200 minimum bets
  • Card counting is legal but casinos may ask you to leave or limit your play if they suspect you are counting. This tool is designed for educational purposes
  • This is a learning tool. Agent B21 helps you understand the math and strategy behind card counting through simulation, not for gambling guidance

How This App Uses Hi-Lo

Agent B21 tracks the running count and true count automatically as cards are dealt from the 6-deck shoe. The count is displayed in the Card Counting panel during gameplay. Bet sizes are adjusted according to the true count bet ramp, and the Monte Carlo simulation engine factors in the actual shoe composition (not just the count) to provide accurate EV-based recommendations.
